Nom gestionnaire de bibliothèque

Description

Votre application de gestion de bibliothèque permet aux utilisateurs de gérer efficacement les livres, les emprunts, et les retours dans une bibliothèque. Elle utilise une interface graphique conviviale avec des fonctionnalités telles que l'ajout de livres, la suppression, la recherche, et bien plus encore.

Fonctionnalités Clés

Gestion des Livres :
Ajouter, supprimer, et modifier les informations des livres (titre, auteur, genre, etc.).
Rechercher des livres par différents critères (titre, auteur, ISBN, genre).
